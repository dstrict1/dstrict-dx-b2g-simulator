## 배포 방법

1. GitHub에서 `b2g-simulator` 레포지토리 생성 (Public)
2. 이 폴더의 파일 4개를 모두 업로드
3. Settings → Pages → Source: **GitHub Actions** 선택
4. 잠시 후 URL로 접속 가능
